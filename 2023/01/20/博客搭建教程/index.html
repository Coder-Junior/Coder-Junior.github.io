博客搭建教程

基础搭建

下载安装node.js

官网安装：https://nodejs.org/en/

安装后验证：windows系统按win+r键，输入cmd打开命令行窗口，输入node -v，若成功显示版本信息则安装成功，如图：

下载安装git

官网安装：https://git-scm.com/download/win

安装后验证：git -v

命令行安装 cnpm

命令：cnpm install -g cnpm --registry==https://registry.npm.taobao.org

安装后验证：cnpm -v

命令行安装hexo

命令：cnpm install -g hexo-cli

安装后验证：hexo -v

配置github

  1. 注册GitHub账号，此步省略；

  2. 新建一个名为：用户名.github.io的仓库，例如github用户名是test，则新建test.github.io的仓库，不会新建仓库的可自行百度，可参考下图：

配置ssh免密登录

  1. 打开电脑文件夹，找到C:\Users\xxx（此处为你的用户名）\.ssh文件夹并删除；
  2. 在C:\Users\xxx文件夹下右键打开Git Bash Here输入命令：ssh-keygen -t rsa -C github注册时的邮件地址，输入后连敲三次回车，出现下图情况代表成功。
  3. 打开新生成的C:\Users\xxx\.ssh文件夹，找到.ssh\id_rsa.pub文件，记事本打开并复制;
  4. 打开github主页，进入个人设置 -> SSH and GPG keys -> New SSH key，粘贴，title随便填，保存即可;
  5. git中输入命令：
    • $ git config --global user.name "xxx"// 引号内为github用户名；
    • $ git config --global user.email "xxx@qq.com"// 引号内为github注册邮箱。

初始化博客

  1. 在电脑的某个地方新建一个名为hexo的文件夹（名字可以随便取），比如我的是E:\xpzsData\hexocode，由于这个文件夹将来就作为你存放代码的地方，所以最好不要随便放；
  2. 在E:\xpzsData\hexocode文件夹下右键打开 Git Bash Here，输入hexo init 初始化；
  3. 在git中执行：
    • hexo g
    • hexo s

更换主题

尽量选用持续维护、文档齐全的主题，这里推荐butterfly主题，官网：Butterfly - A Simple and Card UI Design theme for Hexo，可自行按照官网教程配置博客，内容很多，先完成基本的导航栏设置，细节可以慢慢来。

新建.md文件

有两种方式：

  1. 定位到hexo根目录，右键选择Git Bash Here，执行命令： hexo new 'my-first-blog'则E:\xpzsData\hexocode\source_posts 下生成相关.md文件，并自动生成了带标题、时间的Front-matter；
  2. 自己打开E:\xpzsData\hexocode\source_posts 目录新建.md文件，打开此.md文件，选择“段落”-“YAML Front-matter”，参考ButterFly官方文档中“主题页面”一文，手动填写标题、时间、标签、类别等信息，例如：
